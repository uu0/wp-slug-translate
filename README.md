# WP Slug Translate

一个 WordPress 插件，使用 AI 翻译服务自动将文章 slug（固定链接）翻译为英文，提升 URL 的 SEO 友好度。

## 🌟 功能特点

- **AI 智能翻译**：使用先进的 AI 翻译服务，支持 45 种语言
- **自动 slug 生成**：发布文章时自动将标题翻译为英文，并用连字符分隔单词
- **失败的优雅处理**：翻译失败时自动使用文章 ID 作为 slug
- **支持自定义 API**：兼容 OpenAI 格式的 API 提供商（如 SiliconFlow、DeepSeek 等）
- **模型选择**：支持从可用模型列表中选择合适的翻译模型
- **一键刷新模型**：实时获取和刷新可用的翻译模型列表
- **智能缓存**：模型列表缓存机制，减少 API 调用
- **源语言设置**：支持 45 种源语言
- **双模式运行**：提供第二模式，兼容某些同步插件
- **绿色环保**：停用时自动清理配置选项

## 📦 安装方法

1. 下载并提取 `wp-slug-translate.zip` 到 `wp-content/plugins/` 目录
2. 通过 WordPress 后台的"插件"菜单激活插件
3. 进入"设置"->"WP Slug Translate"进行配置

## ⚙️ 配置说明

### AI 翻译服务配置

在插件设置页面，需要配置以下参数：

#### 1. API Key
- 输入你的 API 访问密钥
- 例如：SiliconFlow、DeepSeek 等平台的 API Key

#### 2. Base URL
- 输入 API 的基础 URL
- 例如：`https://api.siliconflow.cn` 或 `https://api.deepseek.com`

#### 3. Model Name（新增功能）
- **下拉选择**：点击模型下拉列表，选择合适的翻译模型
- **刷新按钮**：点击刷新按钮（🔄）获取最新的可用模型列表
- **自动过滤**：自动过滤掉图片生成模型，只显示聊天和文本模型
- **智能缓存**：模型列表会被缓存以减少 API 调用

**注意**：
- 模型列表会从配置的 API 自动获取
- 建议使用支持中文的模型以获得更好的翻译效果
- 刷新模型列表时保持当前选择的模型（如果该模型仍在列表中）

### 源语言设置

**新增功能**：
- 支持从可用模型列表中选择合适的翻译模型
- 提供一键刷新模型列表功能
- 自动过滤图片生成模型，只保留聊天模型

### 第二模式

运行在第二模式下，兼容某些同步插件。

## 🚀 使用方法

1. 安装并激活插件
2. 在"设置"->"WP Slug Translate"中配置 API 信息和选择模型
3. 选择你的源语言（45 种语言支持）
4. 根据需要启用第二模式
5. 撰写文章并点击"发布"
6. 文章 slug 会自动翻译为英文，单词之间用连字符分隔

**示例**：
- 原标题：如何使用这个WordPress插件
- 自动生成的 slug：`how-to-use-this-wordpress-plugin`

## 🔧 高级功能

### 模型管理

插件提供了智能的模型管理功能：

- **自动获取模型列表**：从 API 端点自动获取所有可用模型
- **模型过滤**：自动过滤图片生成模型，只显示适合翻译的聊天模型
- **实时刷新**：点击刷新按钮即可获取最新的模型列表
- **状态提示**：显示刷新状态和可用模型总数
- **保持选择**：刷新后自动保持之前选择的模型

### 缓存机制

模型列表会被缓存 1 小时，自动减少不必要的 API 调用。点击刷新按钮时会强制更新缓存。

### 错误处理

- **翻译失败**：自动回退到使用文章 ID 作为 slug
- **API 错误**：显示详细的错误信息，方便排查问题
- **无效模型**：提供模型验证和友好提示
